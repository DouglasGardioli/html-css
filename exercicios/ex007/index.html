Principais Formatações

Negrito / Destaque

O Negrito pode ser obtido usando a tag <b>.

Para dar forte importância ao texto use <strong>.

Itálico / Ênfase

Para adicionar itálico a um termo usamos o <i>.

Você consegue dar ênfase a um texto usando o <em>.

Marcar texto

Utiliza-se a tag <mark> para marcar um texto.

Texto grande e pequeno

Temos um texto grande e um texto pequeno neste parágrafo.

Texto deletado

Podemos marcar um texto excluido que deve ser lido, mas não deve ser considerado.

Texto inserido

Podemos marcar um texto inserido para dar um ênfase e mostrar que ele foi adicionado depois.

Existe também a tag <u> que deixa o texto sublinhado (sem semântica).

Texto Subscrito

Você pode usar a tag <sub> para deixar um texto Subscrito como em H2O.

Texto Sobrescrito

Também existe o texto Sobrescrito, utilizando a tag <sup> você pode fazer como em 22.
